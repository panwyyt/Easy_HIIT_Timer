## How to Use

1. **Configure Your Workout**
   - Adjust work time using the +/- buttons (default: 30s)
   - Set rest time between intervals (default: 15s)
   - Choose number of rounds (default: 6)
   - View total workout time in the pill at the top

2. **Start Training**
   - Tap "Start Workout" to begin
   - 5-second countdown to prepare
   - Follow the visual and audio cues
   - Progress ring shows time remaining

3. **Manage Session**
   - Pause/Resume using the bottom button
   - Stop workout with the red square button
   - Mute/unmute audio with the top-right icon
